# dotfiles

作成時の参考サイト：https://qiita.com/yutkat/items/c6c7584d9795799ee164

## 利用方法

1. Cloneする
2. 設定を展開する

```bash
 source ./.bin/install.sh
```

## 設定を追加した時

.gitignoreの対象になっていないかどうか注意する

※ ホワイトリスト方式のため、許可しないとアップロードされない。

（変なシークレットキーが流出するのを防ぐために設定している。）
